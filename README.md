# PropIQ: AI-Powered Real Estate Valuation

PropIQ is a machine learning pipeline designed to predict property values with high precision. Using the Ames housing dataset, this project achieves a 0.8963 R2 score.

## Project Features
* Automated Data Pipeline: SQL-based ingestion and automated feature engineering.
* Advanced ML: Gradient Boosting Regressor with a robust preprocessing pipeline.
* Smart Imputation: Handles missing real estate data without data loss.

## Performance
* Accuracy (R2): 89.63%
* Mean Absolute Error: $17,610.59
